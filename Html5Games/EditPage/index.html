Edit pager

A small bookmarklet that you can use to edit any webpage!

Install

  1. Drag this to your bookmark bar: Edit pager
  2. And you are all set! Click on the bookmarklet to edit a page! If you want to reset the page then just hit reload.

If you want to remove the abilty to edit but keep your changes on the page then drag this to your bookmarks bar: Unedit.

Why would you need this? Maybe if you want to make it look like thats how the page appered or something. Dont ask me!

If you are having problems with the bookmarklet or need help installing it, then please come to my discord server and ask!
